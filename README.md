# 开口英语 · 180天英语改造计划

面向英语初学者的静态学习产品。课程目标不是“打卡180天”，而是用每天留下的听、说、读和个人表达结果，逐步达到稳定B1、熟悉领域接近B2的表现。

## 已包含

- 6个30天阶段、180个逐日主题
- 150节训练课、24次周复测、6次阶段盲测
- 900个不重复的双语高频词块
- 分阶段增长的听力脚本与阅读材料
- 35 / 90 / 180分钟三种训练模式
- 个人日记表达与D+1 / 3 / 7 / 14 / 30间隔复习
- 无台词首听、英文台词查漏和无台词复测
- 浏览器英文朗读、语音识别及原始音频本机保存
- 视频台词点击查词和单词发音
- V2旧版30天学习记录无损迁移
- 学习记录JSON导出

## 本地运行

在本目录执行：

    python3 -m http.server 5190 --bind 127.0.0.1

然后打开 http://127.0.0.1:5190/ 。

语音识别和录音推荐使用最新版Chrome或Edge，并允许麦克风权限。学习状态保存在浏览器的本地存储，原始录音保存在IndexedDB中。
