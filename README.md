# 开口英语

为英语初学者设计的30天开口基础产品，也是六个月交流计划的第一阶段，包含：

- 30天不重复主题与150个核心表达
- 15、30、60分钟三种每日学习模式
- 真实操作驱动的每日完成标准
- VOA官方初级英语沉浸视频
- 每周口语复测与30天阶段复测
- 浏览器英文朗读与语音识别
- 个人表达库与间隔复习
- 英文回答长度和真实对话次数记录
- 本地学习进度保存

## 启动

在此目录运行：

```bash
python3 -m http.server 5173
```

然后打开 <http://localhost:5173>。

语音识别推荐使用 Chrome 或 Edge，并允许麦克风权限。
